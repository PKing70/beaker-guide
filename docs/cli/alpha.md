# Beaker alpha

```
usage: beaker alpha [<flags>] <command> [<args> ...]

Alpha commands with limited support

Flags:
  -h, --help     Show context-sensitive help (also try --help-long and --help-man).
  -v, --version  Show application version.
      --debug    Print verbose stack traces on error.
      --addr="https://beaker.org"  
                 Address of the Beaker service.

Subcommands:
  tensorboard-logs --output=OUTPUT <group>
    Sync tensorboard logs for one or more experiments
```
